# Building a SIEM Dashboard Using ELK Stack

## Project Description

This project demonstrates how to build a Security Information and Event Management (SIEM) dashboard using the ELK Stack (Elasticsearch, Logstash, and Kibana). It collects security logs generated by Python scripts, processes them using Logstash, stores them in Elasticsearch, and visualizes them through interactive dashboards in Kibana.

The project helps security analysts monitor system activities, detect suspicious events, analyze log data, and investigate security incidents in real time.

---

## Objectives

- Build a working SIEM dashboard.
- Collect logs using Python.
- Parse logs using Logstash.
- Store logs in Elasticsearch.
- Visualize logs in Kibana.
- Detect brute-force attacks.
- Detect failed logins.
- Detect port scanning.
- Learn ELK Stack architecture.
- Understand SIEM concepts.

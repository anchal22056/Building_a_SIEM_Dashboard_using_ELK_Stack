# Building a SIEM Dashboard Using ELK Stack

## Project Overview

This project demonstrates how to build a Security Information and Event Management (SIEM) dashboard using the ELK Stack.

The dashboard collects logs from multiple systems, processes them using Logstash, stores them inside Elasticsearch, and visualizes security events through Kibana.

---

## Technologies Used

- Elasticsearch
- Logstash
- Kibana

- Docker

- REST API

## Features

## Project Workflow

1. Generate Logs
2. Send Logs to Logstash
3. Parse Logs
4. Store in Elasticsearch
5. Visualize in Kibana
6. Detect Attacks
